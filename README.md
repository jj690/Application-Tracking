# Application Tracking

A system for tracking job applications, interviews, and recruitment progress.

## Features

- Track application status and timeline
- Manage interview schedules
- Store company and position details
- Monitor follow-ups and communications

## Getting Started

1. Clone or access the repository
2. Review the application tracker structure
3. Begin logging applications and updates

## Usage

Use this tool to organize your job search and maintain an overview of all active applications across different companies.

## Notes

### Exposed Ports and Services

- **Port 3000**: Application Tracking API
- **Port 5432**: PostgreSQL Database (Internal only)
- **Port 5678**: N8N Application
- **Port 8080**: Web UI Dashboard
- **Port 9000**: Admin Panel
